Drug search application. The ability to search the list by symptom. Categories of drugs. Basket. Orders.

MVP (Moxy)
RxJava2
Cicerone (Navigation)
Retrofit
Toothpick (DI)
ViewBinding
Room
Glide
Modularization: data, domain, app
